# Build files

This directory contains builds and files from various tools. They build in
different directories without interfering with each other:

```text
build
 ┼ clion-debug
 ├ clion-default
 ├ clion-release
 ├ run-docker-tests-bash
 ├ run-docker-tests-bat
 ├ run-tests-bash
 ├ vscode
 │
 └ dependency-cache
```
